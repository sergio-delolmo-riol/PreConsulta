Política de privacidad

Protección de datos personales

En PreConsulta tratamos los datos personales con responsabilidad y seguridad. Esta política explica qué datos recopilamos, con qué finalidad y cómo los protegemos.

Datos que recopilamos

  • Datos identificativos: nombre, edad, contacto (cuando el usuario los proporciona).
  • Datos clínicos básicos facilitados durante el triaje (síntomas, antecedentes).
  • Metadatos técnicos: fecha/hora de uso y eventuales datos de dispositivo necesarios para soporte.

Finalidad del tratamiento

Los datos se usan para:

  • Facilitar el proceso de pre-triaje y mejorar la atención sanitaria.
  • Comunicación con el usuario respecto a su consulta.
  • Mejoras de servicio y análisis anónimo para calidad y seguridad.

Conservación y seguridad

Los datos se almacenan en servidores seguros y se conservan durante el tiempo necesario para cumplir las finalidades indicadas o según exigencia legal. Aplicamos medidas técnicas y organizativas para proteger la información.
